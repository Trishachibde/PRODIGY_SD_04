# PRODIGY_SD_04
Web Scraping Project 

This project is a Web Scraping application developed using Python as part of my internship Task 4.
The program extracts data from a website and collects information such as product name, price, and rating. The collected data is then stored in a structured CSV file for easy analysis.
In this project, I used Python libraries like requests and BeautifulSoup to access and extract data from web pages. I also used pandas to store the extracted data into a CSV file.
This project helped me understand how web scraping works, how to extract useful information from websites, and how to store data in a structured format.
